```bash
go get github.com/synapse-garden/sg-proto
sg #              \
# -addr 127.0.0.1 \
# -port :12345    \
# -db my.db       \
# -cert cert.pem  \ # (To use SSL / HTTPS)
# -key cert.key   \
# -cfg conf.toml
```

## Config

The `conf.toml` file specifies config options.
